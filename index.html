2. Features

  • Dynamic buildings list.
  • Requirements check based on buildings, government and planet type selected.
  • Resource output, upkeep and total outcome autocalculated.
  • Total time and resource cost required to build the planet.
  • Job list and housing required.
  • Customization panel to add own resource output or upkeep, using absolute values or percentage, and absolute values for jobs too.
  • Import/export planet templates to save your work and share it.
  • Load custom versions. You can create/modify/delete buildings, jobs, planets type and its districts.

Import version:

By default, you can access files from custom version in C:/Users/[YOUR_USERNAME]/AppData/LocalLow/Stellaris Planet Builder Simulator/Stellaris Planet Builder Simulator in Windows.

For Linux and MAC users, you can find this path in the new version panel. If you open that path, you can see a custom folder, initially empty. When you create a new version, it will be stored in ./custom/[VERSION_NAME] folder and each file referenced in the new version panel will be copied there. Note: file extension must be .txt
Note: files names must be the same as required in each field.
